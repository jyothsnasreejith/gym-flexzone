## Notes

1. **Priority Replacements**: Start with high-impact files like Dashboard.jsx, AddOns.jsx, and CategoryManagement.jsx which have many replacements.
2. **Consistent Patterns**: Most text-slate-900 → text-white, text-gray-* → text-secondary or text-muted
3. **Card Backgrounds**: All bg-white → bg-primary-blue for consistency
4. **Border Colors**: All border-slate-* and border-gray-* → border-navy/* variants
5. **Badge System**: Use badge-success, badge-error, badge-info, badge-warning classes for status indicators
6. **Hex Codes**: Convert text-[#101418], text-[#111418], etc. to semantic color names
